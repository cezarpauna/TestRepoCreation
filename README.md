# Local Coffee Shop

A simple React app showcasing a local coffee shop.

## Features
- Stylish design
- Menu with various coffee options

## Installation
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the application.

## Menu
- Espresso
- Latte
- Cappuccino
- Cold Brew
- Pastries

Enjoy your coffee!
