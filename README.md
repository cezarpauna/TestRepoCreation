## Get Started
To run this simple React app, make sure you have Node.js and npm installed. Then run:

```bash
# Clone the repo
git clone https://github.com/cezarpauna/TestRepoCreation.git

# Change directory
dcd TestRepoCreation

# Install dependencies
npm install

# Start the app
npm start
```

Enjoy your coffee!
