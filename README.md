## Getting Started
To run this app, make sure you have Node.js installed on your machine.

1. Clone this repository:
   ```bash
   git clone https://github.com/cezarpauna/TestRepoCreation.git
   cd TestRepoCreation
   ```  
2. Install dependencies:
   ```bash
   npm install
   ```  
3. Start the app:
   ```bash
   npm start
   ```
