## Getting Started
To start this project, clone the repository and run it locally.

### Prerequisites
- Node.js

### Installation
1. Clone the repo
   ```bash
   git clone https://github.com/cezarpauna/TestRepoCreation.git
   ```
2. Install npm packages
   ```bash
   cd TestRepoCreation
   npm install
   ```
3. Run the app
   ```bash
   npm start
   ```
