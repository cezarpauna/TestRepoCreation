# Local Coffee Shop

Welcome to our Local Coffee Shop! Here, we serve the best coffee in town.

## Our Menu
- Espresso
- Latte
- Cappuccino
- Cold Brew
- Pastries

## Installation
To get started with the app, clone this repository and install the dependencies:

```bash
npm install
```

## Usage
To start the app, run:

```bash
npm start
```
