# Local Coffee Shop

Welcome to our Local Coffee Shop! We serve the best coffee in town.

## Our Menu
- Espresso
- Latte
- Cappuccino
- Cold Brew
- Pastries

### Getting Started
To run the app:
1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies: `npm install`
4. Start the app: `npm start`
