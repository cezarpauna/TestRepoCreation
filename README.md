# Local Coffee Shop

## Welcome!

This is a simple React app created for a local coffee shop. Here you can find our menu and learn about our offerings.

## Features
- User-friendly interface
- Stylish design
- Easy navigation

## Menu
- **Espresso**
- **Latte**
- **Cappuccino**
- **Cold Brew**
- **Pastries**

## Getting Started
To get started, clone this repository and run the app using:
```bash
npm install
npm start
```
