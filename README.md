## Getting Started
To get started with this project, clone the repository and run the app:

```bash
# Clone the repo
git clone https://github.com/cezarpauna/TestRepoCreation.git

# Change to the project directory
cd TestRepoCreation

# Install dependencies
npm install

# Start the app
npm start
```
