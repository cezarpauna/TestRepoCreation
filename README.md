# Local Coffee Shop

Welcome to our Local Coffee Shop! We serve the best coffee in town, along with a variety of delicious pastries.

## Our Menu
- Espresso
- Latte
- Cappuccino
- Cold Brew
- Pastries

## Get Started
To get started with this project, clone the repository and run the app.

```bash
npm install
npm start
```
