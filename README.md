## Getting Started
To run this app locally:
1. Clone the repository:
   ```
   git clone https://github.com/cezarpauna/TestRepoCreation.git
   ```
2. Navigate into the project directory:
   ```
   cd TestRepoCreation
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```
