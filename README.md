## Getting Started
To get a local copy up and running follow these simple steps.

### Prerequisites
- npm
- Node.js

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/cezarpauna/TestRepoCreation.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the application
   ```sh
   npm start
   ```
