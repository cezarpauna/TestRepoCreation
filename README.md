# Local Coffee Shop

Welcome to our local coffee shop! This simple React app showcases our delicious menu offerings. 

## Features
- Stylish design
- Menu listing

## Menu
- Espresso
- Latte
- Cappuccino
- Cold Brew
- Pastries

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the app: `npm start`
