# Local Coffee Shop

Welcome to our Local Coffee Shop!

This is a simple React application that showcases our menu and welcomes customers.

## Features
- Stylish UI
- List of coffees and pastries

## Getting Started
To run this app, clone this repository and run:

```bash
npm install
npm start
```

## Menu
- Espresso
- Latte
- Cappuccino
- Cold Brew
- Pastries
