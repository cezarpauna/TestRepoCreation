## Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/cezarpauna/TestRepoCreation.git
   cd TestRepoCreation
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the application**:
   ```bash
   npm start
   ```

Enjoy your coffee!
