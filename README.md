## Getting Started
To get a local copy up and running, follow these simple steps:

1. Clone the repo
   ```bash
   git clone https://github.com/cezarpauna/TestRepoCreation.git
   ```
2. Navigate to the project directory
   ```bash
   cd TestRepoCreation
   ```
3. Install dependencies
   ```bash
   npm install
   ```
4. Start the application
   ```bash
   npm start
   ```
