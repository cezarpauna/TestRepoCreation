### Getting Started
To run this app locally:
1. Clone the repository
```bash
git clone https://github.com/cezarpauna/TestRepoCreation.git
```
2. Navigate to the project directory
```bash
cd TestRepoCreation
```
3. Install the dependencies
```bash
npm install
```
4. Start the development server
```bash
npm start
```

Enjoy your coffee!
