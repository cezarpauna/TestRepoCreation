We serve the best coffee in town, along with delicious pastries.

## Menu
- **Espresso**  
- **Latte**  
- **Cappuccino**  
- **Cold Brew**  
- **Pastries**  

## Getting Started
To run the app locally:
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the app with `npm start`.

Enjoy your coffee!
