We serve the best coffee in town! Come discover our wide variety of beverages and pastries.

## Features
- Beautifully designed interface
- User-friendly menu
- Responsive layout

## Menu
- Espresso
- Latte
- Cappuccino
- Cold Brew
- Pastries

## Getting Started
To run the app:
1. Clone the repository
2. Run `npm install`
3. Start the app using `npm start`
