# Local Coffee Shop

A simple React app showcasing a local coffee shop.

## Features
- Stylish UI
- Responsive design

## Menu
- Espresso
- Latte
- Cappuccino
- Cold Brew
- Pastries

## Getting Started
To run this app:
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the app: `npm start`
