## Get Started
To clone the repo:
```bash
git clone https://github.com/cezarpauna/TestRepoCreation.git
```

To run the app:
1. Navigate to the directory: `cd TestRepoCreation`
2. Install dependencies: `npm install`
3. Start the app: `npm start`

Enjoy!
