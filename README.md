# Local Coffee Shop

Welcome to our Local Coffee Shop! This is a simple React application that showcases our delicious coffee menu.

## Features
- Stylish interface
- Easy to navigate
- Built with React

## Menu
- Espresso
- Latte
- Cappuccino
- Cold Brew
- Pastries

## Getting Started
To run this project locally, clone the repository and run the following commands:

```bash
npm install
npm start
```

Enjoy your coffee!
