## Installation
To run this project locally:
1. Clone the repo:
   ```bash
   git clone https://github.com/cezarpauna/TestRepoCreation.git
   ```
2. Navigate to the project directory:
   ```bash
   cd TestRepoCreation
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
