## Menu

- Latte

- Cold Brew
- Pastries

## How to Run This App
1. Clone the repository: `git clone https://github.com/cezarpauna/TestRepoCreation`
2. Navigate to the project folder: `cd TestRepoCreation`
3. Install dependencies: `npm install`
4. Start the app: `npm start`
