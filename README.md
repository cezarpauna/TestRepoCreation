# Local Coffee Shop

## Welcome to our Local Coffee Shop

This is a simple React application that showcases our coffee shop and the delicious items we offer.

## Features
- Stylish UI
- Simple and clean design
- Responsive layout

## Menu
- Espresso
- Latte
- Cappuccino
- Cold Brew
- Pastries

## Installation
```bash
npm install
npm start
```
