## Getting Started
This is a simple React app showcasing a local coffee shop.

### Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/cezarpauna/TestRepoCreation.git
   ```
2. Navigate to the directory:
   ```bash
   cd TestRepoCreation
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the app:
   ```bash
   npm start
   ```
