# Local Coffee Shop

Welcome to the Local Coffee Shop, where we serve the best coffee in town!

- Stylish design
- Interactive menu
- Responsive layout

- Latte

## Getting Started
To get started, clone the repository and install the dependencies:

```bash
npm install
npm start
```
