## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/cezarpauna/TestRepoCreation.git
   ```
2. Navigate to the project directory:
   ```bash
   cd TestRepoCreation
   ```
3. Install the required packages:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
