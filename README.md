# Local Coffee Shop

Welcome to the Local Coffee Shop! We serve the best coffee in town.

## Our Menu
- Espresso
- Latte
- Cappuccino
- Cold Brew
- Pastries

## Getting Started
To get started with the app:
1. Clone this repository.
2. Run `npm install` to install dependencies.
3. Start the app with `npm start`.
