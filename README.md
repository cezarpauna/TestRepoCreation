### Getting Started
To get a local copy up and running follow these simple steps.

1. Clone the repo:
   ```bash
   git clone https://github.com/cezarpauna/TestRepoCreation.git
   ```
2. Install NPM packages:
   ```bash
   npm install
   ```
3. Run the app:
   ```bash
   npm start
   ```
