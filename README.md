# Local Coffee Shop

Welcome to our Local Coffee Shop! We serve the best coffee in town. Visit us to enjoy a variety of delicious drinks and pastries.

## Menu
- Espresso
- Latte
- Cappuccino
- Cold Brew
- Pastries

## How to Run the App
1. Clone the repo
2. Navigate to the project directory
3. Run `npm install`
4. Run `npm start` to launch the app.
