## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/cezarpauna/TestRepoCreation.git
   ```
2. Change directory to the project folder:
   ```bash
   cd TestRepoCreation
   ```
3. Install the necessary packages:
   ```bash
   npm install
   ```
4. Start the app:
   ```bash
   npm start
   ```

Enjoy your coffee!
