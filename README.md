## Getting Started
To get started with this React app, clone the repository:
```
git clone https://github.com/cezarpauna/TestRepoCreation.git
```
Then navigate into the directory and install the dependencies:
```
cd TestRepoCreation
npm install
```
Run the app:
```
npm start
```
