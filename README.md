## Getting Started
To run the app locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/cezarpauna/TestRepoCreation.git
   ```
2. Navigate into the directory:
   ```bash
   cd TestRepoCreation
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```
