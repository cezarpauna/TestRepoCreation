## Getting Started
To get a local copy up and running, follow these simple steps:

1. Clone the repo:
   ```bash
   git clone https://github.com/cezarpauna/TestRepoCreation.git
   ```
2. Change to the directory:
   ```bash
   cd TestRepoCreation
   ```
3. Install NPM packages:
   ```bash
   npm install
   ```
4. Start the React app:
   ```bash
   npm start
   ```
