## Getting Started
To get started with this React app:
1. Clone the repository:
   ```bash
   git clone https://github.com/cezarpauna/TestRepoCreation.git
   cd TestRepoCreation
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the app:
   ```bash
   npm start
   ```
