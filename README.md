## Getting Started
To get a local copy up and running follow these simple steps.

### Prerequisites
- Node.js
- npm (Node Package Manager)

### Installation
1. Clone the repo
   ```bash
   git clone https://github.com/cezarpauna/TestRepoCreation.git
   ```
2. Navigate to the project directory
   ```bash
   cd TestRepoCreation
   ```
3. Install NPM packages
   ```bash
   npm install
   ```
4. Run the app
   ```bash
   npm start
   ```
